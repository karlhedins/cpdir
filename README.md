# cpdir

copy files/folders utility wrappers

# Setup

```bash
yarn install
```

# Run

Copy folder A into B, while preserving duplicate files in B already there

First check the contents of `testfolder1` and `testfolder2`

Then run

```bash
yarn start
```

Now see what happened in the folders
